## 📖 Descripción

Este proyecto consiste en una prueba de concepto jugable basada en el videojuego clásico 'Ice Climber'.  
El objetivo principal es replicar sus mecánicas fundamentales mediante el desarrollo de una versión funcional reducida, enfocada en los niveles 1 y 3 del juego original.

El desarrollo se centró en identificar y reproducir los sistemas que definen la experiencia del juego, priorizando la jugabilidad por sobre la fidelidad completa al contenido original.
